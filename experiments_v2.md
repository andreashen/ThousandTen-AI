## 核心公式验证

### 得分 = 消行次数 × 20.3
通过30局统计验证：
- 平均消行 162.5 次 → 平均得分 3290.9 分
- 得分/消行比 = 20.3（理论值 20.0，误差 1.5%）

### 推论
- 要达到 10000 分需要约 **493 次消行**
- 要达到 50000 分需要约 **2463 次消行**

## 新增基础设施

### 1. 可复现基准系统
**文件**: `src/ai/benchmark.py`

**功能**:
- 基于随机种子的可复现测试
- 固定种子的积木序列生成
- 多种子对比测试支持

**使用**:
```python
from src.ai.benchmark import run_multi_seed_benchmark
results = run_multi_seed_benchmark([42, 123, 456], "v1.4")
```

### 2. 策略版本库
**文件**: `src/ai/strategies.py`

**功能**:
- 管理所有策略版本
- 支持版本号复现
- StrategyFactory 工厂模式

**使用**:
```python
from src.ai.strategies import StrategyFactory
dm = StrategyFactory.create_decision_maker("v1.4")
```

### 3. 50种子对比测试
**文件**: `run_benchmark_comparison.py`

**功能**:
- 50个固定种子
- 跨策略对比
- JSON 结果导出

**运行**:
```bash
python3 run_benchmark_comparison.py
```

## 性能优化历程

### 阶段1: 消行优先（12x）
- 提交: `01d2cb8`
- 核心改进: 消行潜力评分 `(filled-6)^2`
- 效果: 122 → 1478

### 阶段2: Clear Priority（1.7x）
- 提交: `431e98f`
- 核心改进: 动作分类 + 序列评估
- 效果: 2060 → 4152 max

### 阶段3: Edge Bonus（1.4x）
- 提交: `0855e4e`
- 核心改进: 边缘放置奖励 `* 2.0`
- 效果: 2977 → 6423 max

### 阶段4: Enhanced Weights（1.7x）
- 提交: `05bb5d9`
- 核心改进: 权重精细调优
  - clear_bonus: 100 → 200
  - future_eval: 1.5 → 2.0
  - edge_bonus: 1.0 → 2.0
- 效果: 5172 → 13177 max 🏆
