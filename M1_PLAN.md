# M1 里程碑实施计划

> **重要提示**: 本文档包含 M1 阶段的完整任务清单和技术规格。新接手的 Agent 或新会话应首先阅读此文，然后查看当前任务状态（通过 TaskList 工具），确保快速上手。

## 项目概述
为 ThousandTen（10x10 积木消除游戏）开发一个纯逻辑的 Python AI Bot，能够独立运行并输出游戏结果，具备 Pygame 可视化界面用于人工验证效果。

## 技术栈
- Python 3.10+
- 游戏引擎: 纯 Python + numpy（可选）
- 可视化: Pygame
- 测试: pytest

## 关键技术点

### 积木类型（全部包含）
- **基础积木**: 1x1, 1x2/1x3 (横竖), 2x2/3x3, L形块
- **M2 积木**: 1x4, 1x5 (横竖)

### 1x4/1x5 长条特殊处理
长条积木会显著增加策略复杂度，AI 需要特别注意：
- 提前为长条预留连续空间
- 避免制造难以利用的"破碎"空地
- 长条的多行/列消除潜力很高，优先级应该提升

### 性能目标
- 单步决策时间 < 1s
- 支持实时可视化（不影响 AI 决策性能）

### M1 不包含
- ❌ 前端对接
- ❌ 复杂的搜索树算法（先验证启发式效果）
- ❌ 机器学习方案
